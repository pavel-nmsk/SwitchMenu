# 🎮 Switch Menu

Десктопное приложение для Windows для просмотра и управления коллекцией игр Nintendo Switch.



## 📋 Описание

Программа сканирует папку с играми, отображает их в виде списка и позволяет просматривать информацию о каждой игре: текстовое описание, обложку и скриншоты геймплея в виде слайд-шоу.



## 🎯 Основные функции

### Просмотр коллекции
- Сканирование папки `Switch` и отображение всех игр в виде списка
- Визуальные индикаторы статуса для каждой игры (кликабельны):
  - 🔵 **Синий** — есть описание (`Description.txt`)
  - 🟢 **Зелёный** — есть обложка (`cover.jpg/png`)
  - 🟡 **Жёлтый** — есть геймплей (`game*.jpg/png`)

### Просмотр информации об игре
- 📖 Обложка
- 🎮 Слайд-шоу из скриншотов (с настраиваемой скоростью)
- 📝 Текстовое описание
- 📜 Бегущая строка для длинных названий

### Редактирование данных
- ✏️ Редактирование описания (встроенный текстовый редактор)
- 🖼️ Замена обложки
- ➕ Добавление новых скриншотов

### Навигация
- 📂 Открытие папки с игрой в проводнике
- 🖱️ Прокрутка списка и картинок колёсиком мыши

## 🚀 Установка и запуск

### Для разработки
```bash
pip install pillow python-docx
python switch_menu.py
Сборка .exe
bash
pyinstaller --onefile --windowed --name "SwitchMenu" --icon="icon.ico" switch_menu.py
Готовый файл: dist/SwitchMenu.exe (~15-20 МБ). Не требует установки Python.

💾 Настройки
Файл menu.ini хранится в папке пользователя:

text
C:\Users\<Имя_пользователя>\switch_menu.ini
ini
[Settings]
auto_scan = True          # Авто-сканирование при запуске
last_path = G:/Switch     # Последний выбранный путь
slide_interval = 3000     # Частота смены картинок (мс)
📦 Вшитые ресурсы
Все ресурсы вшиты в код (base64) и не требуют внешних файлов:

Файл	Назначение
icon.ico	Иконка окна и .exe
Z.jpg	Фото автора
fon.jpg	Фоновое изображение
